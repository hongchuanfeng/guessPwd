# 🎯 猜密码游戏 - 完整需求文档

## 一、项目概述

### 1.1 游戏简介
一款融合逻辑推理与心理博弈的密码破解游戏。玩家需要在限定时间内，根据有限提示猜出隐藏在数字方格后的正确密码。游戏通过逐步增加难度，考验玩家的推理能力、观察力和运气。

### 1.2 视觉风格
- **整体风格**：赛博朋克 + 极简主义，深色主题配合霓虹渐变
- **主色调**：深紫黑背景 `#0a0a1a`，霓虹紫 `#9b59b6`，霓虹青 `#00d4ff`，警告红 `#ff4757`，成功绿 `#2ed573`
- **字体**：等宽字体显示密码格（体现数字感），圆润无衬线字体用于UI
- **动效**：密码输入时有按压反馈，成功/失败有视觉冲击动画

### 2.2 密码验证逻辑

**正确提示（每次验证后显示）：**
1. `数字正确但位置错误` - 显示绿色数字 ✓
2. `数字完全正确` - 显示金色高亮 ✨
3. `数字不存在` - 显示灰色/隐藏

**验证反馈动画：**
- 每个输入格在验证时有轻微弹跳动画
- 正确数字位置正确：格子边框变金色 + 放大效果
- 正确数字位置错误：格子边框变绿色 + 摇晃效果
- 错误数字：格子变红 + 震动效果

## 三、难度系统（核心设计）

### 3.1 难度等级总览

| 等级 | 密码位数 | 提示类型 | 时间限制 | 其他限制 |
|:---:|:---:|:---|:---:|:---|
| 1 | 4位 | 颜色提示 | 无 | 纯数字 |
| 2 | 4位 | 颜���提示 + 数字范围 | 120秒 | 纯数字 |
| 3 | 5位 | 颜色提示 + 范围 + 奇偶 | 90秒 | 含重复数字 |
| 4 | 6位 | 颜色提示 + 范围 + 位置提示 | 60秒 | 含重复数字 |
| 5 | 6位 | 全部提示 + 相邻提示 | 45秒 | 含重复数字 |
| 6 | 7位 | 全部提示 + 质数提示 | 40秒 | 特殊数字 |
| 7 | 8位 | 全部提示 + 计算提示 | 35秒 | 含运算符号 |
| 8 | 8位 | 全部提示 + 序列提示 | 30秒 | 特殊数字 |
| 9 | 9位 | 全部提示 + 高级提示 | 25秒 | 含运算符号 |
| 10 | 10位 | 全部提示 + 终极提示 | 20秒 | 综合挑战 |

### 3.2 各难度详细说明

---

#### 🟢 等级1：入门
- **密码形式**：4位纯数字（0000-9999）
- **提示类型**：颜色提示（类似 Wordle 风格）
  - 绿色：数字正确且位置正确
  - 黄色：数字正确但位置错误
  - 灰色：数字不存在
- **时间限制**：无
- **解锁条件**：默认开放

---

#### 🟡 等级2：数字迷宫
- **密码形式**：4位纯数字
- **提示类型**：颜色提示 + 每个正确数字显示范围
  - 范围提示：`[数字] 在 1-5 之间` 或 `[数字] > 5`
- **时间限制**：120秒
- **特殊规则**：
  - 倒计时结束自动失败
  - 超时扣分加倍
- **解锁条件**：等级1通关

---

#### 🔵 等级3：镜像挑战
- **密码形式**：5位数字，可能包含重复
- **提示类型**：颜色提示 + 范围 + 奇偶提示
  - 奇偶提示：`[位置] 是奇数位` / `[数字] 是偶数`
- **时间限制**：90秒
- **特殊规则**：
  - 密码中可能出现重复数字（如 1123）
  - 需要考虑重复数字的逻辑（如 1 出现几次）
- **解锁条件**：等级2通关

---

#### 🟠 等级4：位置战争
- **密码形式**：6位数字
- **提示类型**：颜色提示 + 范围 + 位置方向提示
  - 位置提示：`[位置] 的数字比 [位置] 大` / `[位置] 是最大/最小的`
- **时间限制**：60秒
- **解锁条件**：等级3通关

---

#### 🔴 等级5：邻居之争
- **密码形式**：6位数字
- **提示类型**：颜色提示 + 全部基础提示 + 相邻提示
  - 相邻提示：`[位置] 和 [位置+1] 是连续数字` / `没有连续数字`
- **时间限制**：45秒
- **特殊规则**：
  - 可能完全没有连续数字
  - 可能有多组连续数字
- **解锁条件**：等级4通关

---

#### 🟣 等级6：质数迷局
- **密码形式**：7位数字
- **提示类型**：全部提示 + 质数提示
  - 质数提示：`[位置] 是质数位` / `密码包含 N 个质数`
- **时间限制**：40秒
- **特殊规则**：
  - 密码中可能包含质数（2,3,5,7）
  - 某些位可能是合数
- **解锁条件**：等级5通关

---

#### ⚫ 等级7：数学家
- **密码形式**：8位数字+运算
- **提示类型**：全部提示 + 计算提示
  - 计算提示：`前四位数字之和=XX`
  - `[位置1] × [位置2] = XX`
  - `相邻两位相加结果`
- **时间限制**：35秒
- **特殊规则**：
  - 密码由数字组成，但提供数学计算结果作为提示
- **解锁条件**：等级6通关

---

#### ⚪ 等级8：数列密码
- **密码形式**：8位数字
- **提示类型**：全部提示 + 序列提示
  - 序列提示：`密码包含等差/等比数列`
  - `某三位数字是斐波那契`
  - `存在递增/递减序列`
- **时间限制**：30秒
- **特殊规则**：
  - 可能存在数学序列模式
  - 观察序列可缩小范围
- **解锁条件**：等级7通关

---

#### 💜 等级9：大师推理
- **密码形式**：9位数字
- **提示类型**：全部提示 + 高级推理提示
  - 暗示密码可能是电话号码尾号格式
  - 暗示可能是日期格式
  - 数字分布密度提示
- **时间限制**：25秒
- **解锁条件**：等级8通关

---

#### 🌟 等级10：终极挑战
- **密码形式**：10位数字（综合）
- **提示类��**：全部提示 + 最终极提示
  - 极少量模糊提示
  - 强烈依赖玩家推理和排除法
- **时间限制**：20秒
- **特殊规则**：
  - 挑战所有前面学到的技巧
  - 运气成分降低，纯靠推理
- **解锁条件**：等级9通关

### 4.2 视觉元素规格

**密码输入格：**
- 尺寸：60px × 80px
- 圆角：12px
- 默认状态：深色背景 `#1a1a2e`，紫色边框 `#9b59b6`
- 输入状态：发光边框效果（box-shadow: 0 0 20px #9b59b6）
- 正确位置：金色边框 `#ffd700` + 脉冲动画
- 正确数字：错误位置：绿色边框 `#2ed573` + 摇晃动画
- 错误数字：红色边框 `#ff4757` + 震动动画

**数字键盘：**
- 按钮尺寸：56px × 56px
- 按钮样式：圆角矩形，带渐变背景
- 按下效果：缩小至95% + 颜色加深
- 按钮间间距：8px

**提示面板：**
- 背景：半透明深色 `rgba(26, 26, 46, 0.9)`
- 边框：1px 霓虹紫渐变
- 提示项左对齐，带图标区分类型

### 4.3 动效规格

**密码格输入动画：**
- 数字填入：从上方滑入 + 淡入（200ms）
- 删除：缩小消失（150ms）

**验证动画序列（总时长800ms）：**
1. 0-200ms：所有格子轻微放大
2. 200-500ms：根据结果变色
3. 500-800ms：错误格子震动，正确格子脉冲

**成功通关动画：**
- 格子依次变金色（100ms间隔）
- 粒子爆炸效果
- 显示成绩卡片

**游戏结束失败动画：**
- 格子全部变红
- 屏幕轻微震动
- 显示正确答案 + 重试按钮

### 5.3 提示生成算法

```javascript
function generateHints(password, userGuess, level) {
  const hints = [];
  
  // 基础颜色提示（所有等级）
  const colorHints = getColorHints(password, userGuess);
  hints.push(...colorHints);
  
  // 等级2+：范围提示
  if (level >= 2) {
    hints.push(...getRangeHints(password, userGuess));
  }
  
  // 等级3+：奇偶提示
  if (level >= 3) {
    hints.push(...getParityHints(password));
  }
  
  // 等级4+：位置关系提示
  if (level >= 4) {
    hints.push(...getPositionHints(password));
  }
  
  // 等级5+：相邻数字提示
  if (level >= 5) {
    hints.push(...getAdjacentHints(password));
  }
  
  // ...以此类推，根据等级增加提示类型
  
  return hints;
}
```
